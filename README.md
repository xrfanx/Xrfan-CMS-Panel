# 📖 About

Xrfan CMS Panel is a modern Admin Dashboard built with **React** and **Vite**.

This project was created to practice building scalable React applications, routing, dashboard UI design, reusable components, charts, data tables, and responsive layouts.

It simulates a CMS (Content Management System) where users and products can be managed through a clean and modern interface.

- 📊 Interactive Dashboard

# 🚀 Installation

Clone the project

```bash
git clone https://github.com/xrfanx/Xrfan-CMS-Panel.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```

Deploy

```bash
npm run deploy
```
